# Notes App

A small single-page notes app where the user can create, edit, and delete notes. All notes save to the browser's localStorage so they persist across reloads.

## Stack

- HTML, CSS, and JavaScript only
- No frameworks (no React, no Vue, no Tailwind, no Bootstrap)
- No build step
- No backend, no database
- Data stored in browser localStorage
- Will be deployed to GitHub Pages

## File structure

Keep everything in these three files at the project root:

- `index.html` — page structure
- `style.css` — all styling
- `app.js` — all JavaScript logic

Do not create additional files (no separate component files, no utility modules) unless I specifically ask for them.

## Rules

- Do not add any libraries or dependencies without asking me first
- Do not introduce a build step
- Keep the code readable — I am still learning, so prefer clarity over cleverness
- When suggesting design choices, explain them briefly so I can learn
- Default to vanilla JavaScript; do not reach for jQuery or similar

## How to run

Open `index.html` directly in a browser. No build, no server needed.
